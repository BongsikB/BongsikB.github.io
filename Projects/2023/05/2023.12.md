## <p align="center">📆 12/1</p>

#### 🎄!

### 🚨 The argument type 'Future<List<fetchData>?>' can't be assigned to the parameter type 'void Function()?'.

```dart
// 💩
onPressed: MyFunction.fetchData(),
```

```dart
// 👏
onPressed: () => MyFunction.fetchData(),
```

- 위는 바로 실행되고, 밑은 이벤트가 일어날 때 마다 실행되는 차이다.
- 매우 기본적인 실수 😮‍💨
