# 💡 Object

📎 배열 Array : 여러개의 `데이터`가 `순서`를 가지고 나열 된 집합 `[]` 대괄호

💡 객체 Object : `Key` 값과 `Value`값을 가진 `프로퍼티 집합`

## { Key : Property }

Key 값과 Value 값을 쌍으로 가지며, 배열과는 다르게 순서가 중요하지 않습니다.

## { Key : Property `,` Key : Property }

쉼표를 사용하여 구분합니다.

## 📌 객체리터럴 Object Literal

```Java script
let person1 = {
  name: '김익명',
  bloodType: 'A형',
  mbti: 'ENFP',
  favoriteCoffee: '아이스아메리카노'
}
```
